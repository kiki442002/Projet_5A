STM32CubeMX Software V4.13.0 / 1 February 2016
Main Changes:
  • Added Linux support.
  • Added support of code generation, clock and power consumption calculation to support the new part numbers of the STM32F0, STM32F3, STM32F4, STM32F7 and STM32L0 series.
  • Added control over order of initialization functions in generated code. In the project settings, the user can choose to generate a CMSIS-Pack description file (gpdsc) to integrate with other IDEs.
