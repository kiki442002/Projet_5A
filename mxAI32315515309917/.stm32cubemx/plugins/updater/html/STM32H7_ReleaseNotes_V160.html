STM32CubeH7 Firmware Package V1.6.0 / 02-Dec-2019
Main Changes
  • Add support of STM32H7A3xx, STM32H7A3xxQ, STM32H7B3xx, STM32H7B3xxQ, STM32H7B0xx and STM32H7B0xxQ new devices
  • BSP drivers developed in line with STM32Cube BSP drivers development guidelines (UM2298)
  • HAL, LL, CMSIS drivers updates
  • Middleware updates : FreeRTOS, FATFS, STM32_NetworkLibrary, OpenAmp, TouchGFX
  • Utilities updates : Lcd_Trace, Basic_GUI, Common
  • Projects updates
