STM32CubeMX Software V4.22.0 / 12 July 2017
Major release
  • Added support for code generation using the low-layer libraries for the STM32L1 Series.
  • The chip view of the pinout tab can be rotated and flipped.
  • In the MCU selector: added the possibility to search for MCUs approaching the entered criteria (close selector).
  • In the pinout view: the user can define several signals on a pin but use them at different time in his code.
