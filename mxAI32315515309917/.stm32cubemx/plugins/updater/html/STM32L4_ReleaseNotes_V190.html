STM32CubeL4 Firmware Package V1.9.0 / 25-August-2017
Main Changes
  • Update of STM32CubeL4 Firmware package to add support for STM32L4R5xx, STM32L4R7xx, STM32L4R9xx, STM32L4S5xx, STM32L4S7xx and STM32L4S9xx devices.
    • New DSI, OctoSPI (OSPI), GFXMMU and LTDC peripherals .
    • Higher RAM size to improve graphic capabilities.
  • Middlewares updates
    • New FatFs R0.12c library
    • New STemWin V5.40 library
