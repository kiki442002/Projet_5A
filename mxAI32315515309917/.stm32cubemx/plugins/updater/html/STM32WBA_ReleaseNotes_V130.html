STM32CubeWBA Firmware Package V1.3.0 / 07-February-2024
Main Changes
Official Release of STM32CubeWBA Firmware package supporting STM32WBA52xx and STM32WBA55xx devices
  • Connectivity
    • Adding support of BLE/Thread and BLE/Zigbee Concurrent Modes.
    • Bluetooth Low Energy new features: Connection Subrating / Channel Classification enhancement / Periodic Advertising Enhancement
    • Bluetooth Low Energy compliancy with TCRL2023-1.
    • Bluetooth Low Energy Audio: Telephony Media Audio Profile : Call Terminal / Unicast Media Receiver / Call Gateway / Unicast Media Sender
    • OpenThread: stack update to SHA-1 : 8bc25042ba5cde20605eec2a329c0dff575303f4 (07/07/2023)
    • Zigbee : Support of GreenPower proxy feature / Adding of additional information in the traces showing useful parameters like link key, cluster in use, end points.
    • Application : BLE and Zigbee project are now supporting SMPS power supply by default.
  • Drivers
    • CMSIS Device Drivers supporting STM32WBA52xx and STM32WBA55xx devices
    • HAL/LL Drivers available for all peripherals
    • BSP Drivers supporting NUCLEO-WBA52CG, NUCLEO-WBA55CG and STM32WBA55G-DK1 boards
  • Middlewares
    • MW ST OpenBL V6.1.0 support
    • MW AzureRTOS v6.2.0 support
    • MW SBSFU/TFM support
  • Projects
    • NUCLEO-WBA55CG
    • Update Applications demonstrating BLE, Zigbee, Thread and MAC 802.15.4 capabilities
    • Update Examples LL demonstrating IP peripherals with use of push-button to enable internal pull-up
    • NUCLEO-WBA52CG
    • Update Examples LL demonstrating IP peripherals with use of push-button to enable internal pull-up
    • STM32WBA55G-DK1
    • Add new applications demonstrating BLE Audio capabilities : BLE_Audio_TMAP_Unicast_Server, BLE_Audio_TMAP_Unicast_Client
    • Update applications demonstrating BLE, Zigbee capabilities
