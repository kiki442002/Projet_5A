STM32CubeMX Software V4.10.0 / 28 August 2015
Main Changes:
  • Added support of code generation, clock and power consumption calculation for STM32F4 series new part numbers.
  • Import of an existing configuration into an MCU of the same series, imports the FreeRTOS configuration and the power consumption calculator data.
  • Added new tab "User Constants" to add and manage needed constants.
  • Default mode for GPIO EXTI: interrupt mode with a rising edge.
  • Additional parameters in RTC configuration window for the time and alarm initialization.
