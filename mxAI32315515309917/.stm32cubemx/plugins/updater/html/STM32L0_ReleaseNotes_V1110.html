STM32CubeL0 Firmware Package V1.11.0 / 30-October-2018
Main Changes
  • Introduced STM32CubeL0 Value Lines.
    • Support of STM32L010xB, STM32L010x8, STM32L010x8, STM32L010x6, STM32L010x4, STM32L010x4 devices.
    • Addition of new projects running for STM32L010xx devices on NUCLEO-L010RB new board.
  • Major update of HAL/LL drivers to:
    • Support new L0 Value line devices.
    • Support HAL callback registration feature.
    • Fix known defects and add several enhancements implementation.
  • Update of FW projects following changes in latest version of HAL/LL and Middlewares.
    • Upgrade to use STM32 TouchSensing Library V2.2.0.
    • Upgrade to use STM32 USB Device Library V2.5.1.
    • Upgrade to use FreeRTOS V10.0.1 (ST modified 20180813).
    • Upgrade to use FatFS R0.12c (ST modified 20171117).
