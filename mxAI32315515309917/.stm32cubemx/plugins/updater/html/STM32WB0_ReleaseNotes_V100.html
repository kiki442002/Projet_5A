STM32CubeWB0 Firmware Package V1.0.0 / 05-June-2024
Main Changes
First Release of STM32CubeWB0 Firmware package supporting STM32WB0x devices
  • Connectivity
    • Bluetooth Low Energy stack
  • Drivers
    • Support CMSIS version v5.9.0
    • CMSIS Device Drivers supporting STM32WB0x devices
    • HAL/LL Drivers available for all peripherals
  • Middlewares
    • STM32_BLE
    • FatFS R0.15 and FreeRTOS V10.5.1
  • Projects
    • STM32_BLE Applications
    • Templates HAL/LL for KEIL and IAR
    • Examples and Examples_LL
    • 2.4 GHz RADIO Demonstrations
