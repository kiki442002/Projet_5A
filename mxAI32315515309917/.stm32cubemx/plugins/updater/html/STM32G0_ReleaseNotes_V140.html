STM32CubeG0 Firmware Package V1.4.0 / 03-November-2020
Main Changes
  • Add support of STM32G0C1xx/B1xx/B0xx devices (CMSIS/HAL drivers)
  • Add support of STM32G061xx/51xx/50xx devices (CMSIS/HAL drivers)
  • Add new middlewares: Openbootloader, USB devices & USB host
  • [FreeRTOS] Support of new R0.12c (ST modified 20191011)
  • [FatFs] Support of new v10.3.1_20200831
  • [USBPD] Support of new USBPD core library (v3.2.0)
  • [USBPD] Support of new USBPD devices library (v3.1.0)
  • [Utilities] Update to latest GUI_INTERFACE & TRACER_EMB version
  • Add New BSP driver for STM32G0C1E-EV board
  • Add new FW examples/application/demo for STM32G0C1E-EV & NUCLEO-G0B1RE boards
  • Add support of STM32CubeIDE for all devices/projects
