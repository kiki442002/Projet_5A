STM32CubeWB Firmware Package V1.16.0 / 08-February-2023
Main Changes

Maintenance Release for BLE, 802.15.4, Thread and Zigbee updates

Porting IAR toolChain 9.20.1.

  • Threadupdates
    • TxPower API support added.
  • Zigbee updates:
    • Update default ZB_NWK parameter and add new configuration (ZB_NWK/ZB_APS).
  • BLE updates
    • BLE GATT caching and Enhanced ATT supported (certified BLE 5.3 - TCRL 2022-1).
    • Application has now the possibility to have scan operation in background with respect to advertising & Connection slave.
  • MAC 802.15.4 updates
    • LowPower application optimization.
    • Improve robustness in case of overloaded 802.15.4 network.
  • BLE/MAC Static updates
    • Improve robustness in case of overloaded 802.15.4 network.
  • BLE/Zigbee Dynamic updates
    • Improve robustness when switching from BLE to Zigbee.
