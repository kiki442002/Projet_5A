STM32CubeF0 Firmware Package V1.2.0 / December 10, 2014
Main Changes
  • Official release of STM32CubeF0 (STM32Cube for STM32F0 Series), introducing STM32F070x6/xB and STM32F030xC derivatives
  • Full set of examples/application/demo running for device
  • STM32F070xB on STM32F070RB-Nucleo board
  • Middleware components update, mainly to fix compilation issues under Linux & keep C/C++ compliance
  • Examples/application/templates/demo update for compatibility
  • MDK-ARM projects converted to µvision 5.
