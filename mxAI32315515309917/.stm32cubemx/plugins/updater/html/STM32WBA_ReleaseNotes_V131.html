STM32CubeWBA Firmware Package V1.3.1 / 22-March-2024

Official Patch Release of STM32CubeWBA Firmware package supporting STM32WBA52xx and STM32WBA55xx devices
This patch release V1.3.1 has to be installed on top of V1.3.0 patch release (based on V1.3.0).

    • Productive RF-PHY settings.

  • Middlewares
    • Middlewares Third Party updates
      • MW SBSFU/TFM support
        • mbed-crypto V2.28.7
        • mcuboot V1.7.2.21
        • tfm V1.3.0.10
  • Projects
    • Update Applications SBSFU/TFM
