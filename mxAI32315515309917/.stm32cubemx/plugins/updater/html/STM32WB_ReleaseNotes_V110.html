STM32CubeWB Firmware Package V1.1.0 / 15-April-2019
Main Changes
  • Add Customer Key Storage (CKS) support.
    • You can refer to AN5185: ST firmware upgrade services for STM32WB Series.
  • Add capability to change the Coprocessor Wireless Binaries Over The Air (OTA).
  • Add a new version of the Firmware Upgrade Service (FUS V1.0.1) to support CKS feature.
    • The FUS V1.0.1 is compatible with the Coprocessor Wireless Binaries provided inside this package (V1.1.0).
    • The FUS V1.0.1 is not compatible with the Coprocessor Wireless Binaries provided inside the previous package (V1.0.0).
