STM32CubeWB Firmware Package V1.14.1 / 19-July-2022

Main Changes

Patch Release for BLE and Thread updates:
  • General updates:
    • Wireless copro binaries corrected to workaround corner case with FUS v1.2.0 that may impact application using OTA to upgrade the copro wireless firmware.
  • BLE updates:
    • Ability to scan only on selected channels.
    • Ability to store Gatt database in NVM upon request.
    • Option provided reducing GATT information size in NVM.
    • Ability to synchronize with additional beacon.
    • Ability to deactivate SMP blacklist mechanism.
  • Thread updates:
    • Thread 1.1 GRL pre-certification passed for FTD and MTD.
    • Thread 1.2 GRL pre-certification passed for FTD only.
    • Fix several OT API wrapper issues.
