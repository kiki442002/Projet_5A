STM32CubeMX Software V4.11.0 / 15 October 2015
Main Changes:
  • Added support of code generation, clock and power consumption calculation for the new part numbers of the STM32L0 series.
  • The power consumption calculator is improved with data interpolation, extended voltage range and max ambient temperature calculation.
  • User defined labels are now generated in the code.
  • In the clock configuration a new button has been added to trigger automatic clock issue resolution.
