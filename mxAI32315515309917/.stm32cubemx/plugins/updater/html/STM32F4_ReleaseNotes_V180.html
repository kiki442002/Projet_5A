STM32CubeF4 Firmware Package V1.8.0 / 14-August-2015
Main Changes
  • Official release to support STM32F469xx / STM32F479xx / STM32F410Cx / STM32F410Rx / STM32F410Tx devices
  • Fix known defects and several enhancements implementation
  • Add new drivers for DSI and LPTIM peripherals
  • Add new BSP drivers for STM32469I_EVAL / STM32469I-Discovery boards
  • Upgrade Middlewares (USB, StemWin, PolarSSL, PDM Audio, FreeRTOS, FatFS) to newer versions
  • Full set of examples for STM32469I_EVAL/STM32469I-Discovery boards
