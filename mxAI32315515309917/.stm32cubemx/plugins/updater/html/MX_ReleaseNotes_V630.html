STM32CubeMX Software V6.3.0 / 02-July-2021
Major release
  • Added the support for the STM32U5 Series:
    • Added the support for the STM32U5xxxx new device
    • Added the support for boards based on devices in the STM32U5 Series:
      • B-U585I-IOT02A
      • NUCLEO-U575ZI-Q
      • STM32U575I-EV
    • Possibility to create projects without or with Arm® TrustZone® security
    • Added the native integration of Azure® RTOS midlleware ThreadX, FileX, LevelX, NetX Duo, and USBX in STM32U5 Series
  • Added the support for new boards: STEVAL-STWINKT1B, STM32MP157D-DK1 and STM32MP157D-EV1
  • Added the support for new features such as license click-through and thread-safe locking support.
