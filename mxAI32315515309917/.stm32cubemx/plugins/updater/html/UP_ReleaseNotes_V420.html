STM32CubeUpdater Software V4.2.0 / April 28, 2014
Enhancements
  • Possibility to activate automatic check of updates at logon.
Main Features
  • Downloading and updating STM32Cube firmware packages allowing the download from www.st.com of the MCU firmware package required for the development of the user application.
  • Self-updates allowing the user to keep the STM32CubeUpdater up-to-date.
