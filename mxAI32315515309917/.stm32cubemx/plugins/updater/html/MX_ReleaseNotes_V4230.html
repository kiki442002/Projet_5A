STM32CubeMX Software V4.23.0 / 20 October 2017
Major release
  • Added code generation support using the low-level libraries (LL) for the STM32F2 Series, STM32F4 Series, and STM32F7 Series.
  • Added support for the new part numbers in the STM32F7 Series and STM32H7 Series.
  • Added support for the MbedTLS middleware.
