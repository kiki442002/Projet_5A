STM32CubeMX Software V4.3.1 / August 07, 2014
Main Features
  • Release 4.3.1 is a maintenance release of STM32CubeMX 4.3. It mainly fixes missing signals, wrong clock constraints and wrong USB code generation on some STM32F4 MCUs, as well as some GPIO and timer issues on STM32F3 MCUs and STM32F0 MCUs.
