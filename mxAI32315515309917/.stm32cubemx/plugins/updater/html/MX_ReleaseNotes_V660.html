STM32CubeMX Software V6.6.0 / 24-June-2022
Major release
  • Added the support for the new STM32MP131, STM32MP133, and STM32MP135 microprocessors
  • Added the support for one new STM32MP135 board
  • Added the support for one new part numbers in the STM32WB Series
  • Added the support for one new part numbers in the STM32L4 Series
  • Added the support for CAD resources in the MCU and MPU Selector and the ST Tools tab.
