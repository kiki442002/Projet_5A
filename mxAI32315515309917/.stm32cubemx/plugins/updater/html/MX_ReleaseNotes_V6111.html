STM32CubeMX Software V6.11.1 / April-2023
Patch release :
  • Patch release to fix these issues :
  • An error occurs when switching the timebase source from SysTick to a timer for the STM32C0 series
  • A compilation error occurs with the MX_FLASH_Init() function for the STM32H5 series
  • STM32CubeMX V6.11.0 has lost the JTAG (5 pins) debug mode for all the STM32H7 series but the STM32H7Rx/7Sx microcontrollers
  • STM32CubeMX V6.11.0 has lost three modes on ADC3 for all the STM32H7 series but the STM32H7Rx/7Sx microcontrollers
  • An issue affects ADC oversampling when writing OVSR bits for all the STM32H7 series but the STM32H7Rx/7Sx microcontrollers
  • STM32CubeMX V6.11.0 is not generating the call to the HAL_PWREx_EnableVddA() function for the STM32U5 series
