STM32CubeMX Software V4.4.0 / September 19, 2014
Main Features
  • Support of code generation, clock and power consumption calculation for the STM32L1 series.
  • MCU selector additional filters allowing to filter by memories and number of I/Os.
  • Board selection can be done based on board peripherals.
  • The clock tree now automatically proposes a solution based on the frequencies entered.
