STM32CubeMX Software V4.20.0 / 28 February 2017
Main Changes:
  • Added support of code generation, clock and power consumption calculation for STM32F0, STM32F1, STM32L0 and STM32L4 Series new part numbers.
  • Added support for libjpeg on the STM32F4 Series.
  • Added support for security IPs.
  • Added support for hardware JPEG on STM32F7.
  • Added code generation on the Low Level Libraries for the STM32F3 Series.
  • Analog signals on the same pins can now be shared.
