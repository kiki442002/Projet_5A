STM32CubeWB Firmware Package V1.13.2 / 15-February-2022
Main Changes

  • BLE updates:
    • Introducing following new binary Full stack extended: stm32wbxx_BLE_Stack_full_extended_fw.bin including BLE extended advertizing features
    • Introducing following new binary HCI layer extended: stm32wbxx_BLE_HCILayer_extended_fw.bin including BLE extended advertizing features
    • Compared to previous delivery, the binary stm32wbxx_BLE_Stack_full_fw.bin is linked to modified Basic stack library with included Phy update(2Mb/s) and additional beacon
    • BLE Binaries FWs are compliant to BLE5.2 certified features TCRL2021-1
  • Zigbee updates
    • Zigbee stack patches in order to solve R22 security vulnerability reported by the CSA (Security Incident Number: 2021-ZP-0401)
    • Fix Manufacturer ID management : Node Descriptor Response Manufacturer ID can be set using ZbApsSet. apsManufacturerId is no more overwritten when calling ZbStartup
  • BLE Zigbee Static updates
    • Fix memory leak on transition from Zigbee to BLE
    • Following application has been added: /P-NUCLEO-WB55.Nucleo/Applications/BLE_Zigbee/BLE_Zigbee_Static
  • BLE-Thread Dynamic Concurrent Mode
    • Correct low power consumption issue
