STM32CubeMX Software V4.6.0 / January 20, 2015
Enhancements :
  • Support of code generation, clock and power consumption calculation for for all the STM32 series, added support for STM32F1 series, STM32F0 and STM32F4 new part numbers.
  • Automatic clock resolution for the whole clock tree.
  • Custom third party code can be generated.
