STM32CubeG4 Firmware Package V1.0.0 / 12-April-2019
  • First official release of STM32CubeG4 Firmware.
    • CMSIS Device, HAL & LL drivers supporting STM32G431/41xx, STM32G471xx, STM32G473/83xx and STM32G474/84xx devices.
    • BSP for Nucleo and Evaluation Boards of STM32G4 series .
    • Rich set of project examples and applications to allow user to quickly build any firmware on a given board; projects can be easily configured and generated with CubeMX application.
