STM32CubeF4 Firmware Package V1.15.0 / 17-February-2017
Main Changes
  • Add the support of the Low layer driver(LL).
  • Add the support of the STM32F413H-Discovery board.
  • Fix known defects and several implementation enhancement.
  • Update Middleware versions for : FreeRTOS, LibJPEG, LwIP and FatFS.
  • Add the support of mbedTLS Middlware.
