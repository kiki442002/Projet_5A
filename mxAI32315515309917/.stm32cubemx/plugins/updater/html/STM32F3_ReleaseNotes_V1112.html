STM32CubeF3 Firmware Package V1.11.2 /10-November-2020
Main Changes
  • Patch release to fix known defects and enhancements implementation..
  • Remove register keyword to be compliant with new c++ standard:
    • The register storage class specifier was deprecated in C++11 and removed in C++17.
  • Update STM32F3-Discovery Demo and BSP example to support new MEMS:
    • gyroscope component **I3G4250D**.
    • accelerometer component **LSM303AGR**.
  • Update All MDK-ARM Templates projects to support new ARM compiler V6.
