STM32CubeMX Software V4.9.0 / 10 July 2015
Main Changes:
  • Added support of code generation, clock and power consumption calculation for STM32L4 series new part numbers.
  • Import of an existing configuration into a MCU of the same series.
  • Recursive mutexes can be created in the FreeRTOS configuration.
  • Configuration report now contains the clock tree, active IP, NIVC, GPIO and DMA configuration.
  • Clock tree can be reset to its default value.
  • FIFO threshold and burst size constraints are managed in the DMA configuration.
