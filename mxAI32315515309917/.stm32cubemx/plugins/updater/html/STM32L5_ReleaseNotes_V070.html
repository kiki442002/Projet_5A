STM32CubeL5 Firmware Package V0.7.0 / 07-October-2019
Main Changes
  • Fourth Beta release of CMSIS Device and HAL/LL drivers for STM32L552xx and STM32L562xx devices (RM0438 revision 4)
  • Fourth Beta release of BSP drivers for NUCLEO-L552ZE-Q (MB1361 revision C) and STM32L552E-EV boards (MB1372 revision C)
  • Third Beta release of BSP drivers for STM32L562E-DK board (MB1373 revision C)
  • Set of legacy (option bit TZEN=0) and TrustZone (option bit TZEN=1) project templates for EWARM, MDK-ARM and STM32CubeIDE, projects examples for EWARM and MDK-ARM
    • See Projects\STM32CubeProjectsList.html for all available projects
    • TrustZone enabled projects name ends with "_TrustZone" (except TFM application)
    • Warning: TrustZone is disabled by default on STM32L5 HW boards: use STM32CubeProgrammer to update the Option bytes prior to download and execution
  • TFM application integration on STM32L562E-DK
