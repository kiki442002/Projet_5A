STM32CubeMX Software V4.24.0 / 19 January 2018
Major release
  • Added support for the STM32F0 Series, STM32F3 Series, STM32F4 Series, STM32F7 Series, STM32L0 Series, STM32L1 Series, and STM32L4+ Series new part numbers.
  • Added support for STM32F1 code generation using the low-level libraries (LL)..
  • Added support for the TouchSensing and PDM2PCM middleware.
  • Added framework to add additional software components to the project.
